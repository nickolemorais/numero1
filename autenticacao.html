Gerando Token

! Passo a Passo estilo Wiki

Para gerar o token de acesso é necessário utilizar a API de autenticação conforme imagem:

1. Abra o postman.

2. Acesse a API de autenticacao.

3. Insira o client key e o client secret.

4. Clicque em send.

5. Copie o token gerado.
