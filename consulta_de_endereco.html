Consulta por texto

Necessário ter o AddressID gerado na API de Autenticacao para utilizar a API de Consulta de Endereço conforme imagem:

1. Abra o postman.

2. Acesse a API de autenticacao.

3. Insira o client key e o client secret.

4. Clicque em send.

5. Copie o token gerado.
